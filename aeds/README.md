# AEDS - Algoritmos e Estruturas de Dados

## Objetivo

Reunir modulos educacionais voltados para o estudo de AEDS II. Os modulos
abordam estruturas de dados, algoritmos de manipulacao, complexidade e
propriedades, sempre de forma interativa e voltada para prova.

## Modulo atual

### [reavaliacao-aeds-2/](reavaliacao-aeds-2/) - Reavaliacao AEDS II

**Estado:** primeira versao funcional e modulo principal de AEDS no momento.

Modulo interativo para estudo de estruturas de dados, com visualizacoes,
simulacoes, exercicios de codigo e atividades conceituais. A experiencia atual
combina campanha de dominios, oficina de codigo, treino continuo, caderno
adaptativo de erros, visualizacoes de estruturas e simulado final.

Dominios iniciais:

- Doidona;
- Arvore TRIE;
- Arvore AVL;
- Arvore normal/binaria;
- Somatorios;
- Algoritmos de ordenacao.

As instrucoes para instalar, executar, testar e gerar build ficam no README do
modulo atual: [`reavaliacao-aeds-2/README.md`](reavaliacao-aeds-2/README.md).
